# ecommerce-data-analysis
This is a Microsoft Excel data analysis project exploring a large e-commerce dataset. Built relationships using Power Pivot, analyzed KPIs with Pivot Tables, and generated business insights on sales, customers, products, regions, and payment methods.
This project presents a comprehensive sales analysis of a large-scale e-commerce dataset using Microsoft Excel. The goal was to transform raw transactional data into actionable business insights by answering key business questions related to sales performance, customer behavior, product performance, geographic trends, and payment methods.

Here is the link to the complete dataset : https://www.kaggle.com/datasets/mmohaiminulislam/ecommerce-data-analysis

The dataset follows a star schema data warehouse design, consisting of one fact table and multiple dimension tables (Customers, Products, Stores, Time, and Transactions). Relationships were created using Excel's Data Model, enabling analysis across multiple tables through Pivot Tables and Pivot Charts.

# Project Objectives
Analyze overall sales performance and key business KPIs.
Identify top-performing products and suppliers.
Understand customer purchasing behavior.
Evaluate sales across different regions.
Analyze sales trends over time.
Examine payment method usage and transaction values.
Generate business recommendations based on data-driven insights.

- Business Questions Answered

This analysis answers 40 business questions, including:

Sales Performance
- What is the total revenue generated?
- How many total units of products were sold?
- What is the average order value (AOV)?
- Which products generated the highest revenue?
- Which products sold the highest quantity?
- Which products generated the lowest sales?
- What percentage of total revenue does each product contribute?
- Which suppliers generated the most revenue?
- Which manufacturing countries contribute the most sales?
- What is the revenue trend over time?

Customer Analysis
- How many unique customers made purchases?
- Who are the top 10 customers by total spending?
- Which customers purchased the greatest quantity of items?
- Which customers have made only one purchase?
- What is the average spending per customer?
- Which customers generate the highest lifetime value?
- How evenly is revenue distributed among customers?
- Are there customers who consistently buy high-priced products?

Product Analysis
- Which product categories (or items) are most profitable based on revenue?
- Which products have the highest average selling price?
- Are products being sold at prices different from their listed prices in the item table?
- Which suppliers provide the best-selling products?
- Which products have the lowest sales despite high prices?
- Which products should potentially be discontinued due to poor performance?

Geographic Analysis
- Which division generates the highest revenue?
- Which district has the highest sales?
- Which upazila has the highest transaction volume?
- Which regions have the highest average order value?
- Which regions sell the largest quantities of products?
- Are there regional preferences for certain products?

Time-Based Analysis
- Which year generated the highest revenue?
- Which quarter performs best in terms of sales?
- Which month has the highest revenue?
- Which week records the most transactions?
- Which day of the month has the highest sales?
- At what hour of the day are sales highest?

Payment Analysis
- Which payment method is used most frequently?
- Which payment method generates the highest revenue?
- Which banks process the highest transaction values for card payments?
- Is there any relationship between payment method and average order value?
